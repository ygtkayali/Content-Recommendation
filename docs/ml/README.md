# ML Team Docs

This folder contains ML-team-specific planning and requirements.

## Scope
- Similarity retrieval and ranking.
- NLP intent parsing and constraint extraction.
- Recommendation quality evaluation.

## Files
- `requirements.md` — ML functional and quality requirements.
- `plan.md` — phased implementation plan for the ML track.
- `dataset-strategy.md` — dataset sourcing, normalization, and quality checks.
- `hybrid-architecture-planning.md` — research and decision framework for hybrid scoring.
- `data-layout.md` — canonical data folder structure and artifact naming/versioning rules.
